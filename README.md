## 📖 Overview

The **Advanced Library Management System** is a web-based application developed using **PHP**, **MySQL**, **HTML**, **CSS**, **JavaScript**, and **Bootstrap**.

This system helps libraries manage books, members, and transactions efficiently.  
It also includes **barcode generation and scanning** functionality to simplify book tracking and improve overall management.

---

## ✨ Features

✅ Add / Update / Delete Books  
✅ Manage Library Members  
✅ Book Issue & Return System  
✅ Barcode Generation  
✅ Barcode Scanner Integration  
✅ Search Books  
✅ Transaction Management  
✅ Export Records

## 🚀 Installation & Setup

### Step 1: Clone Repository
```bash
git clone https://github.com/Tanzilanaikwadi/Advance-Library-Management-System-Using-Barcode-Scanner.git
```

### Step 2: Move Project Folder
Move project folder into:
```bash
C:\xampp\htdocs\
```

### Step 3: Import Database
Import `database.sql` into phpMyAdmin.

### Step 4: Start Server
Start:
- Apache  
- MySQL  

### Step 5: Run Project
Open browser and visit:

```bash
http://localhost/library
```





## 📌 Usage

- Add books to the library database  
- Generate barcodes for books  
- Scan barcodes using scanner/webcam  
- Issue books to members  
- Return books and update records
